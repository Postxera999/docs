## Page Overview

The Venue Detail page provides complete information about each entertainment venue within Galaxy: Galaxy Arena, G Box and Broadway Theatre. It includes venue visuals, seating plans, special offers, location details and key information related to the venue. This page references data from the **Entertainment Venue content type** within the Entertainment module, integrated with Contact, Offer, and Event Schedule entities as defined in the CCMS.

## 6 Floor Plan Button

```Plain
Global_ACTION_FloorPlanButton
```

### Key Function

- The **Floor Plan Button** allows users to view the venue’s seating or layout map image as configured in the CCMS.
    
- It is used on the **Venue Detail** page to provide a quick reference for visitors to locate areas within the entertainment venue.
    

Please refer to the global component documentation for display details. View](/01_Global_Components/ FloorPlanButton)

  

### Field List

| Fields           | Description                                                                                  |
| ---------------- | -------------------------------------------------------------------------------------------- |
| floor_plan_image | The image to be displayed when the user clicks the floor plan button at the top of the page. |

### Display Logic

- Visible when a **Floor Plan Image** is uploaded under the _Entertainment Venue_ content type.
    
- Hidden automatically if no Floor Plan image is available.
    

### Interaction Logic

- On tap, opens the full floor plan in a full page image viewing mode which supports zoom and pan interactions.
    
- For larger layouts, the view switches to horizontal orientation to improve visibility.

## 9 Upcoming Show List

```Plain
ENT_LIST_UpcomingShowsList
```

### Key Function

- The **Floor Plan Button** allows users to view the venue’s seating or layout map image as configured in the CCMS.
    
- It is used on the **Venue Detail** page to provide a quick reference for visitors to locate areas within the entertainment venue.
    

Please refer to the global component documentation for display details. Link

  

### Field List

This component references data listed in the **Entertainment Event content type** within the Entertainment module.

| Fields                   | Description                                                                                |
| ------------------------ | ------------------------------------------------------------------------------------------ |
| show_id                  | A unique internal identifier used to link Entertainment Events with related content types. |
| venue_id                 | Identifies the Entertainment Venue of the Entertainment Event                              |
| tile_image               | A single hero image is associated with each Entertainment Event.                           |
| title_text               | The title text of the Entertainment Event.                                                 |
| show_times_specification | The show time entries for each Entertainment Event.                                        |

### Display Logic

- Each Entertainment Event is presented as an individual show card with fixed width and height.
    
- The `show_id` list displayed in the component is configured by the `venue_id` of the Entertainment Event.
    
- `title_text` appears in maximum of 2 lines and is truncated if the character count exceeds the card width
    
- `show_times_specification` appears in a maximum of 1 line and is truncated if the character count exceeds the card width
    
- By default, a maximum of 4 cards are displayed on the page. If the `show_id` list exceeds 4, a full width [View More] button appears with the number of hidden shows.
    

### Interaction Logic

- On tap, each card links to its respective Enterainment Event page.
    
- If the `show_id` list exceeds 4 and [View More] button is displayed, tapping View More will expand the full list and the button will disappear
    

  

### CMS Field Specification

See Entertainment Event detail page for details on field input. [View](/02_Page_Documentation/M06_03_Entertainment_Event_Detail)
