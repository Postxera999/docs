## 2.  Search Suggestion List


```Plain
TRANS_LIST_SearchSuggestionList
```

### Key Function

This component dynamically displays search term suggestions and/or relevant content shortcuts as the user types, accelerating the search process.

### Visual Reference

<FigmaEmbed url="https://www.figma.com/design/EHB41JQIomI0iwPWzaXAkw/-Dev--NGMP-UXUI-Designs?node-id=7460-40871&t=TmrwzwgJVnbh1PL8-0" />

### Field List

| Fields          | Description                                   |
| --------------- | --------------------------------------------- |
| `merchant_type` | Category/type of merchant or related listing. |
| `merchant_name` | Display name of the linked merchant.          |

### Display Logic

- The `SearchSuggestionList` appears immediately below the `SearchBar` when the search input is focused and/or non-empty.
- Suggestions are updated in real-time based on the user's partial input.
- Provide results based on **fuzzy search**.

### Interaction Logic

- Tapping on any suggestion item populates the `SearchBar` with the suggestion text and automatically triggers a search for that term, navigating to or updating the `SearchResultBlock`.

## 3.  SearchResultBlock
  
```Plain
TRANS_LIST_SearchResultBlock
```

### Key Function

This component displays the list of content or carparks.

### Visual Reference

<FigmaEmbed url="https://www.figma.com/design/EHB41JQIomI0iwPWzaXAkw/-Dev--NGMP-UXUI-Designs?node-id=7729-17966&t=jk8IFspVTdwcceRA-4" />

### Field List

| Fields                      | Description                                                                           |
| --------------------------- | ------------------------------------------------------------------------------------- |
| `carpark_name`              | Displays the official or common name of the parking facility.                         |
| `carpark_description`       | Displays the official or common name of the parking facility.                         |
| `carpark_label`             | A categorical label or tag indicating a specific feature, status, or type of parking. |
| `carpark_merchant_distance` | The distance between a merchant and a carpark.                                        |

### Display Logic

- The `SearchResultBlock` appears when a search query is submitted (by tapping a suggestion).
- Results are displayed in a scrollable list, typically in a card-based format, ordered by
- Sort merchants based on the user’s selected merchant, in order of distance from nearest to farthest.
- When no search results are found, display the **No Result** screen.

### Interaction Logic

- Tapping on a result item navigates the user to the corresponding carpark detail page. Check [M13_07 Carpark](/02_Page_Documentation/M13_Transportation/M13_07_Carpark) for details。
