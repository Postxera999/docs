1. ## Landing Video Background
    

```Plain
SIGNUP_BINDING_VISUAL_LandingVideoBackground
```

Full-screen visual layer for the signup landing page. Displays a looped video (with image fallback) behind the Signup Card.

---

**Display Logic**

- Fills the entire viewport as the page background.
    
- Plays the configured video on load; muted and looped by default.
    
- Falls back to the poster image when autoplay is restricted.
    
- Always used together with **Signup Card** in the full-page signup layout (card rendered above this background).
    

---

### CMS Field Specification

|   |   |
|---|---|
|Specification Item|Details|
|Field Type|URL (video source)|
|Example|/assets/signup-bg.mp4|
|Data Source|CMS Signup Configuration Table|

7. ## Terms & Conditions Modal
    

```Plain
SIGNUP_BINDING_CTNR_Terms&ConditionsModal
```

### Key Function

Modals containing required legal consent statements. User must accept before login proceeds.

---

### Field List

|   |   |
|---|---|
|Field|Description|
|terms_title|Modal title text (e.g., “Terms & Condition”)|
|consent_1_text|First consent statement|
|consent_2_text|Second consent statement|
|note_text|Gray note block below checkboxes|

---

### Display Logic

- Bottom sheet with rounded corners; close “✕” on the top-right.
    
- CTA button stays disabled until all required checkboxes are checked.
    
- WeChat Login: CTA button is for One-Tap login.
    
- Phone Number Login: CTA button continues for mobile input.
    

### Interaction Logic

- Tapping the checkbox toggles state.
    
- Once mandatory fields are checked → CTA becomes active.
    
- Close icon returns user to signup background.
    

---
