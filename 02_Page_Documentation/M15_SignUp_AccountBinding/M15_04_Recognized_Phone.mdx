3. ## RecognizedMembershipCard
    

```Plain
GLOBAL_CTNR_RecognizedMembershipCard
```

### Key Function

Displays the **detected GPC membership details** so that the user can confirm the account before binding.

### Used on

- RecognizedPhone (M15_04)
    
- Any flow where an existing GPC membership is auto-detected
    

### Visual Reference

---

### Field List

|   |   |
|---|---|
|Field|Description|
|gpc_tier|Current GPC tier (e.g., “GPC Black”).|
|galaxy_life_tier|Matching Galaxy Life tier (e.g., “Galaxy Life Diamond”).|
|masked_name|Masked member name (e.g., “L** D****”).|
|gpc_id|Member ID (e.g., 9232 1232).|
|gpc_phone|Linked phone number (e.g., +853 2323 1234).|
|card_image|Image of the membership card.|

### Display Logic

- Shown when **exactly one** GPC membership is matched to the phone number.
    
- If matching fails or multiple matches exist, a different scenario page/modal should be used.
    

### Interaction Logic

- The card itself is read-only.
    
- Confirm/reject actions handled by **NotMyAccountButton** and **BindNowPrimaryButton**.

4. ## NotMyAccountButton
    

```Plain
SIGNUP_BINDING_ACTION_NotMyAccountButton
```

### Key Function

Let the user indicate that the recognized GPC membership might **not** be theirs.

### Used on

- RecognizedPhone (M15_04) – directly under the membership card
    

---

### Visual Reference

---

### Field List

|   |   |
|---|---|
|Field|Description|
|label|Button label text. Default: “Not My Account”.|

---

### Display Logic

- Shown whenever a membership card is displayed.
    
- Hidden if no membership is recognized.
    

### Interaction Logic

- On tap, opens the **“Not My Account” modal**, which:
    
    - Explains that the GPC membership may not belong to the user.
        
    - Allows navigation to **Bind Another Account** flow (member ID + PIN).
        
    - Allows skipping for now.

10. ## BindNowPrimaryButton
    

```Plain
GLOBAL_ACTION_BindNowPrimaryButton
```

### Key Function

Primary CTA for the user to **confirm binding** of the recognized GPC membership and activate the tier match.

---

### Used on

- RecognizedPhone (M15_04) the right button in sticky bar
    
- Other flows requiring a “Bind Now” primary action
    

---

### Visual Reference

---

### Field List

|   |   |
|---|---|
|Field|Description|
|label|Button text (e.g., “Bind Now”).|

---

### Display Logic

- Shown when there is a valid recognized membership card.
    
- May be disabled if binding prerequisites are not met (e.g., missing consent).
    

### Interaction Logic

- On tap:
    
    - Calls backend to bind the recognized GPC membership.
        
    - On success, routes to **Binding Success / Confirmation** page.
        
    - On failure, shows an error message or retry state.

|   |   |
|---|---|
|Specification Item|Details|
|Field Type|Text (single line)|
|Max Length|25 characters|
|Example|Bind Now|
|Data Source|CCMS Binding Configuration (Button Labels)|
