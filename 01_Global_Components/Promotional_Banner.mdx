## Key Functional Features

  

This module presents membership related promotions and/or segmentation promotional offers related to the page in a dynamic, visually compact format. It allows users to see what offers/rebate promotions are applicable to them e.g. Birthday Offer, Rebate Rate, Direct Offers by segment.

  

## Display Logic

- The module is visible only when the user is logged in. When displayed, it includes:
    
    - The offer/benefit card type and corresponding tag (see below Supported Card Types) .
        
    - The offer title and description
        
    - If configured, the expiration of the promotional offer on the top right corner
        
    

## Layout

- Single card: displayed at full width.
    
- Multiple cards: along with other promotional benefit card, each card displayed at 300px width in a scrollable horizontal carousel
