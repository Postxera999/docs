#### Key Functional Features
- Display logic:
    - NG$ Rebate _(default)_: If the merchant offers rebate, by default it will show this tag
    - Special Rebate: If the merchant is offering a special rebate rate, the tag name customizable and configurable in the CCMS and can be tagged to the applicable merchants. e.g. Double Rebate, 1.5X Bonus Points
