The **Login Sticky Button** is a floating global call-to-action prompting users to **log in** to access member-exclusive features. It remains visible on public pages until the user is logged in.

---

## Display Logic

- Floating global CTA for logged out users.
    
- Prompts users to sign in to access member-exclusive bene.

## Key Functional Features

| No. | Feature                | Description                                                      |
| --- | ---------------------- | ---------------------------------------------------------------- |
| 1   | Floating CTA           | Stays visible as user scrolls, improving accessibility.          |
| 2   | Conditional Visibility | Displays only when user is not logged in.                        |
| 3   | Dynamic Label          | Text changes depending on context (e.g., “Log In to View More”). |
| 4   | Global Application     | Shared across all entertainment modules.                         |
