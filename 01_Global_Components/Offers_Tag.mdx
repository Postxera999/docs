## Key Functional Features

- If an Offer Content Type has been linked to the page content in the `CCMS`, the Offer tag will be displayed based on the following states:
    

| **No. of offers linked** | **Background Colour**                             | **Display Logic**                                                  | **Example**                                              |
| ------------------------ | ------------------------------------------------- | ------------------------------------------------------------------ | -------------------------------------------------------- |
| 1                        | White                                             | Displayed using Gold tag, '1 Offer'                                | <img src="/2_Images/M03_Retail/img_5.png" alt="描述文字" />  |
| 2 or more                | White                                             | Displayed using Gold tag, 'X Offers' where X=no. of offers linked  | <img src="/2_Images/M03_Retail/img_6.png" alt="描述文字" />  |
| 1                        | Dark colour detected via dominant colour function | Displayed using Light tag, '1 Offer'                               | <img src="/2_Images/M03_Retail/img_7.png" alt="描述文字" />  |
| 2 or more                | Dark colour detected via dominant colour function | Displayed using Light tag, 'X Offers' where X=no. of offers linked | <img src="/2_Images/M03_Retail/img_8.png" alt="描述文字" />  |
| 1                        | Light colour via dominant colour function         | Displayed using Dark tag, '1 Offer'                                | <img src="/2_Images/M03_Retail/img_9.png" alt="描述文字" />  |
| 2 or more                | Light colour via dominant colour function         | Displayed using Dark tag, 'X Offers' where X=no. of offers linked  | <img src="/2_Images/M03_Retail/img_10.png" alt="描述文字" /> |
