## Key Functional Features

- Displayed in tool bar: The name of the chosen sorting option is displayed in full in the top bar in an active selected state. The user can select the dropdown to switch sorting options.
    
- Sorting pop-up: Implemented as a half-screen popup that slides up from the bottom.
    
- Sorting options: Includes sorting options selectable via radio buttons, one option must be selected at all times.
    
- Confirmation button: Once a sorting option is selected, clicking the 'Confirm' button will apply the selection to the main page.
    

| **Sorting Options**                | **Description**                                                                           |
| ---------------------------------- | ----------------------------------------------------------------------------------------- |
| **Recommended** _(Default)_        | Displays content according to the order determined in the `CCMS`.                         |
| **A–Z** _(Alphabet, English only)_ | If the MP language = English, sorts content alphabetically in ascending order.            |
| **Z–A** _(Alphabet, English only)_ | If the MP language = English, sorts content alphabetically in descending order.           |
| **A–Z** _(Pinyin, Chinese only)_   | If the MP language = Chinese, sorts content alphabetically in pinyin in ascending order.  |
| **Z–A** _(Pinyin, Chinese only)_   | If the MP language = Chinese, sorts content alphabetically in pinyin in descending order. |
| **Newest**                         | Sorts stores by the most recently added entries in the directory.                         |
