The **Body Content** component displays detailed text and media information for detail pages. It supports both plain text and formatted content (e.g., paragraphs, links, and images)

## Key Function

- Displays a detailed write-up of the any content’s description.
    
- Supports rich text input and embedded images.
    

---

### CMS Field Specification

| Field Name        | Description                                      |
| ----------------- | ------------------------------------------------ |
| description_title | Heading for the description section (tab label). |
| description       | Body text or media content of the event.         |

---

### Field Mapping Details

#### `description_title`

| Specification Item | Details                            |
| ------------------ | ---------------------------------- |
| Field Type         | Free text input                    |
| Max Length         | 37 characters                      |
| Input Validation   | Mandatory                          |
| Localization       | Must support all display languages |
| Data Source        | CCMS                               |
| Dependencies       | –                                  |

#### `description`

| Specification Item | Details                            |
| ------------------ | ---------------------------------- |
| Field Type         | Rich text field                    |
| Max Length         | 10,000 characters                  |
| Input Validation   | Optional                           |
| Localization       | Must support all display languages |
| Data Source        | CCMS                               |
| Dependencies       | –                                  |
