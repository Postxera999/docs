## Key Function

The Entertainment Recommendations **Carousel** is a global component used across the Entertainment pages that displays recommended or entertainment events at the bottom of the page. It enhances user engagement by encouraging further browsing of upcoming shows or promotions related to the current content.

| No. | Feature               | Description                                                              |
| --- | --------------------- | ------------------------------------------------------------------------ |
| 1   | Related Event Display | Shows a list of recommended events configured in CCMS.                   |
| 2   | Global Consistency    | Can be shared across multiple entertainment modules for cross-promotion. |
| 3   | Swipe Interaction     | Enables users to horizontally scroll through related items.              |
| 4   | Dynamic Linking       | Each card links to the corresponding Enterainment Event detail page.     |

---

## Display Logic

- Appears when related content or recommendations are configured in the CCMS, or if there are no results in the Entertainment directory search.
    
- Each carousel card displays event image, title, and short description.
    
- Hidden automatically when no related content is available.
    
- Clicking a card navigates users to the linked Entertainment Event detail page.
    
- Supports both manual swipe interactions for carousel navigation.

## CMS Field Specification

| Field Name    | Description                                                  |
| ------------- | ------------------------------------------------------------ |
| Event Title   | Title of the recommended event.                              |
| Event Image   | Image thumbnail or banner used in carousel card.             |
| Event Link    | Link to the related Entertainment Event detail page.         |
| Display Order | Determines order of events within the carousel.              |
| Is Active     | Indicates whether the recommendation is currently displayed. |
