## Key Function

- This is a **popup modal component** that is triggered when the user clicks the [Contact Button](/01_Global_Components/Contact_Button).
    
- It contains information from the Contact Card Specification
    
- The contact card specification should support the following channels:
    
    - CS Hotline (phone number)
        
    - Live WeChat CS (direct to WeChat CS)
        
    - Add Galaxy WeCom (long press QR code)
        
    - Personal Assistant contact number + copy Whatsapp number (if assigned)

- The `contact_card_title` and `contact_card_description` are aligned left in a bounding box of width: 222px to allow for negative space on the right.
    
- Service Hotline
    
    - The `cs_hotline_title` and `cs_hotline_description` is left aligned and the text wraps only if the bounding box width exceeds 222px or if it meets the hotline button (allowing for 8px margin).
        
    - The `cs_hotline_number` is placed in a button style aligning right.
        
- WeChat CS
    
    - The `wechat_cs_title` and `wechat_cs_description` is left aligned and the text wraps only if the bounding box width exceeds 222px or if it meets the hotline button (allowing for 8px margin).
        
    - The `wechat_cs_cta_text` is placed in a button style aligning right.
        
- General WeCom
    
    - Only shown in the Default contact card, if PA is assigned to the user it does not show.
        
    - The `general_wecom_title` and `general_wecom_description` is left aligned and the text wraps only if the bounding box width exceeds 222px.
        
    - `wecom_qrcode` is aligned right
        
- Personal Assistant assigned to user
    
    - Displayed when a PA is assigned to the user and the card style is according to the user's tier
        
    - Each contact option is displayed as it's own separate card
        
        - WeCom: card includes `pa_wecom_title`, `pa_wecom_description` and `wecom_qrcode`
            
        - WhatsApp: card includes `whatsapp_title` and `whatsapp_number`with a copy button next to the number.

## Interaction Logic

- Tapping the `cs_hotline_number` button triggers the **native phone call modal** for users to call directly.
    
- Tapping the **Customer Service on WeChat** button opens the **official WeChat** **live** **chat window**.
    
- Long pressing the `wecom_qrcode` triggers the add WeCom flow.
    
- Tapping the copy button next to the `whatsapp_number` copies the number to the users clipboard.
