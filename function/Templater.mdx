## Main / Directory Page

- **入口**：<模块入口，例如首页卡片、导航入口>  
- **页面标题**："<页面标题>"  

### 主要区域
1. **功能导航区**（Filter / Search / Sort / Quick Filter 等）  
   - **功能逻辑**：
     - Filter: 用户点击触发筛选弹窗 → 选择条件 → 点击确认 → 列表刷新  
     - Search: 用户输入关键词 → 触发搜索逻辑 → 显示匹配结果  
     - Sort: 用户选择排序条件 → 列表实时排序  
     - Quick Filter: 点击不同分类标签 → 更新列表数据  

2. **提示信息区**（用户状态 / 模块提示信息）  
   - **逻辑说明**：
     - 根据用户登录状态或会员等级显示不同提示  
     - 无登录或普通用户显示通用提示  

3. **内容列表区**（列表或卡片形式展示内容）  
   - **内容元素**：
     - 主图 / Logo  
     - 标题 / 名称  
     - 分类 / 类型  
     - 状态 / 可用性  
     - 可选标签 / 活动 / 优惠信息  
   - **功能逻辑**：
     - 点击卡片 → 跳转详情页  
     - 数据动态加载（分页 / 下拉加载 / 异步加载）  
     - 空状态处理：无数据时显示提示文案

- **联系按钮 / 联系弹窗**  
  - 触发逻辑：点击 → 弹窗显示 → 用户操作（拨打电话 / 在线客服 / 关闭弹窗）  
- **分享按钮 / 分享弹窗**  
  - 触发逻辑：点击 → 弹窗显示 → 用户操作（分享到好友/朋友圈/保存图片）  
- **优惠券 / 活动使用逻辑**  
  - 状态管理：未领取 / 已领取未使用 / 已使用 / 已过期
