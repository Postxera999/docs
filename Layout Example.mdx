## 3. Text Formatting

- **Bold**: `**bold**` or `__bold__`
- *Italic*: `*italic*` or `_italic_`
- ***Bold + Italic***: `***bold italic***`
- ~~Strikethrough~~: `~~strikethrough~~`

---

## 4. Lists

### Unordered List
- Item one
- Item two
  - Nested item
    - Deeply nested

### Ordered List
1. First item
2. Second item
   1. Sub-item
   2. Another sub-item

### Task List
- [x] Completed task
- [ ] Incomplete task
- [ ] Nested task
  - [ ] Sub-task
